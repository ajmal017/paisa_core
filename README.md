# paisa_core
(still under development) <br/>
Python library for 5paisa trading APIs.

# To test the library.

- create env file.

```bash
mv .env.template .env
chmod +x .env 
```
- fill out all the variables from app creds you have and user authentication detail and execute the file.
```bash 
./.env
```
- Now you can main.py.
